The module solves the coupled level set and volume of fluid method. Method: The interface tracking method is used to solve volume of fluid. The level set function is used to compute the normal vector which give a better accuracy than using volume fraction. When the new position of interface is determined the redistance for level set will be applied.

    • Cutcell
